# Goal V1 - Monitoring Foundations

## 1. Objective
Build a minimal monitoring setup for web service

The goal is to have:
 - Container resourse monitoring
 - Service availability monitoring
 - Basic alerting

---

## Environment

 - Single Linux Host
 - Containerized services
 - Centralized monitoring system

---

## Components

 - Linux Host VM
 - Monitoring server
 - Monitoring database
 - Web frontend
 - Monitored web service

---

## Success Criteria

The goal is considered complete when:
 - Host CPU, memory and disk metrics are visible via web interface
 - Web service avalability is monitored
 - At least 3 alerts are configured
